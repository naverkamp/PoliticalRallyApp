This website was built using RShiny.

The following data sets were used:

  • New York Times Covid-19 county-level data
  • US Census Bureau Adjacency file
  • US Census Bureau county-level population data
  • simplemaps.com data connecting counties to cities

This website was created as part of a project for BST 260 at Harvard University in the School of Public Health for the MS Biostatistics program. Kirstie Turnbull and Anja Shahu were group members who contributed with their feedback and debugging help.
